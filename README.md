# Random Message Generator

Random Message Generator is a JavaScript app for outputting a random joke message.

## Installation

Use HomeBrew[https://brew.sh/] to ensure that NodeJS[https://nodejs.org/] is installed.

```bash
brew install node
```

## Usage

```bash
node rmg.js
```
